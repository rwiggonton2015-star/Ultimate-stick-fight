# Ultimate Stick Fight - Web Version

A fun 2D side-scrolling stick man fighting game!

## How to Play

### Controls
- **Desktop:**
  - Arrow Keys or A/D - Move left/right
  - Spacebar - Jump
  - Click or F - Attack

- **Mobile:**
  - Use on-screen buttons to move, jump, and attack

## Gameplay
- Fight red stick figure enemies
- Each enemy defeated gives you 50 points
- Each hit gives you 10 points
- Don't let enemies touch you - they deal damage!
- Reach 0 health and it's game over

## Play Online
Open this file in your browser or go to:
